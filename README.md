# Bridge PDE/ODE Project

This project is a bridge PDE/ODE application built with CMake, C++, and Python.

## Requirements

- CMake
- C++
- Python

## How to run

From the project root:

```bash
mkdir build
cd build
cmake --build build ..
```

## Run the code

```bash
./bin/bridge-pde-ode
```
